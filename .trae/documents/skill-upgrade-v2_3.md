**修改后**：
```markdown
### 极速检测模式

**触发关键词**：快速排雷 / 直接测敏感词 / 极速检测 / 快速扫描 / 只查敏感词

当识别到极速检测意图时，允许跳过 @工作流1（账号定位）和 @工作流3（匹配度诊断），直接执行 @工作流2（快速敏感词检测），以节约交互时间。

**极速检测流程**：
1. 无需收集账号信息
2. 直接扫描正文、标题、标签中的敏感词
3. 仅输出：总体结论 + 敏感词检测结果 + 修改建议
4. 不输出：内容定位卡、匹配度分析、自然流诊断、CES优化

```

```
```

### 2.2 补全「视频笔记」检测维度

**位置**：在 @工作流2 的视频笔记检测部分

**修改前**：
```markdown
**视频笔记额外检测项**：
- [ ] **脚本合规性**：视频口播内容是否包含敏感词或违规表达？
- [ ] **字幕敏感词**：字幕文字是否包含违规内容？
- [ ] **画面合规性**：是否包含二维码、微信号、其他平台水印？
```

**修改后**：
```markdown
**视频笔记额外检测项**：

在 @工作流2 中，针对视频笔记必须执行以下检测路径：

1. **口播脚本检测**
   - 是否有高频敏感词（同一敏感词重复出现 ≥3 次）
   - 是否语速过快导致信息密度过高（单句字数 >50 字）
   - 是否有重复性极限用语（如连续使用"最""第一"等）

2. **字幕合规检测**
   - 错别字变体（如"关住"替代"关注"）
   - 极限词（"最好""最强"等）
   - 拼音缩写（"wx""vx"等）
   - emoji替代敏感词

3. **画面合规预警**（AI无法直接检测，需提示用户自查）
   - ⚠️ **二维码**：视频画面中是否出现微信/其他平台二维码？
   - ⚠️ **平台水印**：是否带有其他平台（抖音、快手等）的水印？
   - ⚠️ **裸露风险**：是否涉及过度暴露或暗示性内容？
   - ⚠️ **品牌侵权**：是否未经授权使用知名品牌 Logo？
   
   > 提示用户："以上画面风险需创作者自行检查，AI无法通过纯文本分析视频画面"
```

```markdown
3. **高分标题备选** - 3个优化后的高分标题（痛点型/好奇型/干货型）
4. **CES评分优化策略** - 围绕点赞率、收藏率、评论率、转发率、完播率的设计建议
5. **网感强化** - emoji使用、口语化表达、情绪价值、UGC真实感、信息增量

```

**修改后**：
```markdown
基于前三个工作流的检测结果，输出：

1. **修改建议** - 针对标题、正文、封面、标签的具体优化方案
2. **合规替代表达** - 高风险表达的安全替代版本
3. **高分标题备选** - 3个优化后的高分标题（痛点型/好奇型/干货型）
4. **CES评分优化策略** - 围绕点赞率、收藏率、评论率、转发率、完播率的设计建议
5. **网感强化** - emoji使用、口语化表达、情绪价值、UGC真实感、信息增量
6. **评论区预埋引导** - 基于 CES 算法，生成 1-2 条高赞神评/置顶评论引导文案
7. **发布前CheckList** - 13项最终检查清单

**评论区预埋引导细则**：

基于 CES 算法，主动为用户生成 1-2 条用于发布后创作者自己抢首评的引导文案：

- **类型A：福利领取指引**
  - 示例："整理了一份【XX资料包】，需要的姐妹可以留言，我发你~"
  - 目的：提升评论率，制造互动理由

- **类型B：争议性话题抛出**
  - 示例："有人说这个方法太麻烦，但我用了三个月真的有效，你们怎么看？"
  - 目的：引发讨论，提升评论率和停留时长

- **类型C：解答补充**
  - 示例："补充一下：视频里没说完的【XX细节】，我放在评论区了"
  - 目的：引导用户查看评论区，提升互动深度

**合规约束**：
- 预埋文案 MUST 避免使用"评论1发你""私信领"等高风险导流表达
- 使用合规替代："感兴趣的朋友可以在评论区交流""有问题欢迎交流，看到都会回复"
```

```markdown
## 输出格式（标准化9步结构）

> **Token 预算提示（内部执行规则）**：
> - 当笔记字数较多或问题清单超过 5 项时，MUST 优先输出【高风险】和【中风险】项
> - 将【低风险】和【合规】项合并一句话带过或使用 Markdown 折叠
> - 防止输出阶段超出最大 Token 限制导致回复截断
> - 具体策略：
>   - 高风险/中风险：详细展开，保留完整表格和修改建议
>   - 低风险：一句话总结，如"另有 2 处低风险表达，建议微调"
>   - 合规项：一句话带过，如"其余内容合规，未发现风险"
```
