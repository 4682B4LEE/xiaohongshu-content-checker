# 极速排雷模式

适用：用户只想快速检查敏感词或高风险表达。

## 输出原则

1. **极简输出**：只给结论和命中项，没有风险的内容不输出。
2. **不询问账号信息**：直接检测，不追问定位。
3. **快速决策**：用户一眼知道是否需要修改。

## 输出结构

### 1. 总体结论

**命中项分值**：单项 +1~+4

**总分**：X 分

**等级**：🔴 高风险 / 🟠 中风险 / 🟡 低风险 / 🟢 安全

**发布建议**：建议重写高风险片段后再发布 / 建议修改后发布 / 建议微调 / 可发布，仍建议人工复核

**命中统计**：🔴 X 处 / 🟠 Y 处 / 🟡 Z 处

- 账号匹配度评分
