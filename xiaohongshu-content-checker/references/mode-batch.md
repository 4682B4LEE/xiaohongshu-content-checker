# 批量检测模式

适用：矩阵账号、内容团队需要一次性检测多条笔记。

## 输入分隔协议

系统按以下优先级识别多篇笔记：

1. **`---` 分隔符**：用户用三个短横线分隔每篇笔记（推荐）。
2. **编号标记**：`笔记1`、`笔记2`、`笔记3` 或 `1.`、`2.`、`3.`。
3. **JSON/YAML 列表**：`[{ "title": "...", "content": "..." }, ...]`。

**无法可靠分隔时**：只问一个澄清问题——"请用 `---` 分隔每篇笔记，方便批量检测。"

## 流程

1. 只询问一次账号信息
2. 按分隔协议拆分输入，对每条笔记依次执行风险检测（加载 risk-rubric.md）
3. 输出汇总报告
