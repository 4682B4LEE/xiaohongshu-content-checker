# 小红书笔记敏感词检测与内容定位分析

## 概述

本 Skill 由平台风控专家视角构建，帮助小红书创作者在发布笔记前进行全方位的内容合规检测、自然流阻碍排查与内容质量优化。核心目标是在保障账号安全的前提下，最大化笔记的自然流量潜力与CES评分表现。

本 Skill 涵盖：

1. **账号人设与笔记意图对齐** - 确认内容策略与商业目标的一致性
2. **内容方向检测** - 分析笔记内容与账号定位的一致性
3. **变体敏感词深度检测** - 识别谐音梗、拼音缩写、emoji替代、错别字等隐蔽违规表达
4. **自然流阻碍因素排查** - 诊断匹配度、营销浓度、外部导流风险
5. **CES评分优化** - 基于点赞、收藏、评论、转发、完播率提升内容质量
6. **网感与情绪价值强化** - 确保内容具备UGC真实感与平台调性

## 核心原则

### 精准定位优于盲目检测

小红书平台对人群画像的匹配度要求很高，发布内容与账号定位不匹配会直接影响自然流量。因此，本 Skill 采用"先定位、后检测、再优化"的标准工作流：

1. **先明确账号定位与人设** - 了解账号类型、目标人群、内容方向、人设风格
2. **再确认笔记意图与关键词** - 明确商业目标与核心关键词布局
3. **然后进行敏感词深度检测** - 识别常规敏感词及变体表达
4. **接着排查自然流阻碍因素** - 诊断匹配度、营销浓度、导流风险
5. **最后输出优化方案与发布CheckList** - 提供修改建议、高分标题备选、CES优化策略

### 影响自然流的关键因素

| 因素类别 | 具体表现 | 风险等级 |
|---------|---------|---------|
| 账号定位混乱 | 内容方向不统一，今天发美妆明天发科技 | 高 |
| 人群画像不匹配 | 内容风格与目标受众喜好不符 | 高 |
| 敏感词违规 | 极限用语、违禁词、引流词（含变体） | 高 |
| 营销浓度过高 | 硬广感强、推销意图明显、缺乏真实体验 | 高 |
| 外部平台导流 | 直接或间接引流至微信、其他平台 | 高 |
| 内容质量低 | 搬运、抄袭、无价值内容 | 中 |
| 互动诱导 | 求关注、求点赞、互粉等 | 中 |
| 发布频率异常 | 短时间内大量发布或长期断更 | 中 |
| 标签使用不当 | 标签与内容不符、堆砌标签 | 低 |

## 工作流程（5步骤SOP）

### 步骤一：账号定位与人设确认（必须）

在检测具体内容之前，必须先明确以下信息。采用**分阶段收集**方式，每阶段提供选项供用户选择：

#### 阶段一：账号基础信息（必须）

**1. 账号类型**（单选）
A. 个人号（以个人身份分享）
B. 企业号（以品牌/公司身份）
C. 达人号（有一定粉丝基础的KOL）
D. 其他

**2. 内容领域**（单选，基于TRAE社区用户背景重新排序）
A. 数码工具（TRAE开发的游戏/工具/App推广）
B. 编程教程（代码分享、技术教程）
C. 效率办公（生产力工具、工作流优化）
D. AI应用（AI工具使用、AI绘画/写作等）
E. 开源项目（GitHub项目推广、技术分享）
F. 产品设计（UI/UX、原型设计、设计工具）
G. 职场成长（面试经验、职业规划、技能提升）
H. 生活方式（数码生活、桌面搭建、极客日常）
I. 其他

**3. 账号阶段**（单选，给出具体分级描述）
A. 新手期（0-1000粉丝，刚起步，建立内容方向）
B. 成长期（1000-1万粉丝，有一定粉丝基础，探索变现）
C. 成熟期（1万-10万粉丝，内容稳定，商业合作较多）
D. 头部期（10万+粉丝，行业影响力，品牌合作频繁）

**4. 账号IP/所在地**（选填）
- 格式：城市名（如：北京、上海、杭州）
- 说明：可能影响同城推荐，但权重不高

#### 阶段二：人设定位（必须）

**人设定位**（单选，给出简单描述）
A. 干货型（分享技术教程、使用技巧、经验总结）
B. 测评型（真实体验产品，客观评价优缺点）
C. 情绪共鸣型（分享开发故事、成长经历、心路历程）
D. 好物分享型（推荐实用工具、效率神器、数码好物）
E. 其他

#### 阶段三：笔记内容（必须）

用户提供：
- 标题：
- 正文：
- 标签：
- 封面文案（如有）：

### 步骤二：笔记意图与关键词确认

在用户提供笔记内容后，再确认笔记意图：

#### 1. 笔记意图（单选）
A. 涨粉导向（希望获得更多关注和曝光）
B. 带货转化（推广产品/工具，引导下载/购买）
C. 纯日常互动（分享日常，增强粉丝粘性）
D. 引流私域（引导到微信/社群/公众号）

#### 2. 目标关键词（用户填写1-3个）
- **核心关键词**：笔记必须覆盖的1-3个核心搜索词（影响搜索流量）
- **长尾关键词**：2-5个相关长尾词（提升精准匹配度）
- **话题标签**：计划使用的#话题标签列表
- **关键词布局检查**：核心关键词是否自然出现在标题、正文前50字、标签中

### 步骤三：变体敏感词深度检测

#### 检测类别

**类别1：广告法极限用语（违规风险-高风险）**
- 国家级、世界级、最高级、顶级
- 第一、唯一、首个、首选、首家
- 最好、最强、最优、最先进
- 极致、绝对、永久、100%、零风险
- 全网第一、销量冠军、领导品牌

**类别2：虚假宣传用语（违规风险-高风险）**
- 夸大功效：立竿见影、瞬间见效、神奇效果
- 绝对承诺：100%有效、保证治愈、永不复发
- 医疗宣称：治疗、治愈、疗效、药方、疗程
- 投资承诺：稳赚、保本、零风险、高收益、暴富

**类别3：平台规则敏感词（违规风险-中高风险）**
- 引流词汇：微信、VX、加V、二维码、私信我、加我
- 外部平台：直接提及其他社交平台名称（视情况）
- 诱导互动：求关注、必回关、互粉、互赞、求点赞
- 虚假互动：刷赞、刷粉、买数据

**类别4：行业特定敏感词（违规风险-根据行业变化）**
- 美妆护肤：美白、祛斑、抗衰、祛痘、修复（需资质）
- 保健食品：减肥、瘦身、燃脂、排毒、疗效
- 母婴用品：治疗、预防、保健功效宣称
- 金融理财：理财、投资、赚钱、收益率、保本
- 教育培训：保过、包过、100%就业、保证提分

**类别5：内容质量风险词（流量风险-中风险）**
- 搬运抄袭：直接复制他人内容
- 标题党：夸张标题与内容不符
- 低质内容：无实质价值、纯广告、硬广

#### 流量风险专项检测

以下问题不会直接导致违规，但会严重影响自然流量分发：

| 流量风险项 | 具体表现 | 影响程度 |
|-----------|---------|---------|
| 内容与账号定位偏离 | 笔记主题与账号历史内容或定位不一致 | 高 |
| 标题过于广告化 | 标题像硬广，缺乏真实感和吸引力 | 高 |
| 开头没有痛点 | 前3秒/前50字未抓住用户注意力 | 高 |
| 用户画像不清晰 | 内容面向人群模糊，无法精准匹配 | 高 |
| 标签过泛 | 使用过于宽泛的标签，缺乏精准度 | 中 |
| 过度堆叠关键词 | 关键词生硬堆砌，影响阅读体验 | 中 |
| 缺少真实体验感 | 内容像官方文案，缺乏个人真实感受 | 高 |
| 商业目的过强 | 推销意图明显，用户产生抵触心理 | 高 |
| 内容没有收藏价值 | 信息密度低，用户没有收藏动机 | 中 |
| 话题与目标人群兴趣不匹配 | 选题与目标受众关注点不符 | 高 |

#### 变体敏感词识别（重点）

平台算法已升级至可识别以下变体表达，必须纳入检测范围：

| 变体类型 | 示例 | 识别要点 |
|---------|------|---------|
| **谐音梗** | "薇信"、"加V"、"威信"、"围脖" | 发音相同或相近的替代字 |
| **拼音缩写** | "wx"、"vx"、"pyq"、"tb"、"jd" | 平台名称、联系方式的拼音首字母 |
| **emoji替代** | "加👆"、"戳👉"、"💌我"、"🔍xxx" | 使用emoji替代敏感动作或联系方式 |
| **错别字/形近字** | "关住"替代"关注"、"点瓒"替代"点赞" | 故意使用错别字规避检测 |
| **符号拆分** | "微-信"、"加 我 薇" | 使用符号或空格拆分敏感词 |
| **英文替代** | "wechat"、"follow me"、"DM me" | 使用英文表达敏感含义 |
| **数字谐音** | "520"、"1314"用于诱导互动 | 结合语境判断是否构成诱导 |

### 步骤四：自然流阻碍因素排查

#### 1. 匹配度诊断
- [ ] **账号-内容匹配**：笔记主题是否与账号领域一致？
- [ ] **人设-表达匹配**：内容风格是否符合选定人设？（干货型避免过于情绪化，情绪共鸣型避免过于生硬）
- [ ] **受众-内容匹配**：目标受众是否与账号画像匹配？
- [ ] **意图-形式匹配**：笔记形式（图文/视频）是否适合当前意图？

#### 2. 营销浓度判断
- [ ] **硬广指数**：是否出现品牌名堆砌、价格频繁提及、购买链接暗示？
- [ ] **种草自然度**：推荐是否基于真实使用场景？有无"广告感"过强的问题？
- [ ] **价值密度**：内容是否以信息价值为主，而非纯推销？
- [ ] **UGC真实感**：语言是否口语化、有真实体验细节？是否像真实用户而非官方账号？

#### 3. 外部平台导流风险
- [ ] **直接导流**：是否包含微信号、手机号、二维码、其他平台链接？
- [ ] **间接导流**：是否使用"私信我"、"主页有方式"、"评论区见"等暗示性表达？
- [ ] **平台提及**：是否直接提及其他竞争平台名称（视平台当前规则动态调整）？

#### 4. 封面文案检测（新增）

封面文字是小红书笔记的第一印象，直接影响点击率。检测维度：

- [ ] **一眼看懂**：封面文字是否在0.5秒内能被理解？
- [ ] **避免过度承诺**：是否包含"必看"、"绝了"、"封神"等过度夸张表达？
- [ ] **字数控制**：封面文字是否在8-16字之间？（信息过多会分散注意力）
- [ ] **内容一致性**：封面文案是否与笔记内容高度相关？
- [ ] **目标人群吸引力**：封面是否能吸引目标人群的注意力？
- [ ] **违规词检查**：封面文字是否包含敏感词或违规表达？
- [ ] **阅读场景适配**：封面在小红书信息流中是否清晰可读？

**输出要求**：针对封面文案，提供3个优化版本建议，分别侧重：
- **版本A**：痛点直击型
- **版本B**：好奇引发型
- **版本C**：价值承诺型

#### 5. 评论区与互动引导检测（新增）

评论区是高风险区域，需重点检测：

**高风险表达识别**：

| 风险类型 | 具体表达 | 风险等级 |
|---------|---------|---------|
| 评论区导流 | "评论1发你"、"私信领"、"加我拿" | 高 |
| 主页导流 | "主页有联系方式"、"看主页" | 高 |
| 诱导互动 | "需要的扣1"、"免费送资料包" | 中高 |
| 诱导点赞收藏 | "点赞收藏不迷路"、"先收藏再看" | 中 |
| 诱导交易 | "限时优惠"、"仅限前10名"、"手慢无" | 中 |

**合规替代表达建议**：

| 高风险表达 | 合规替代方案 |
|-----------|-------------|
| "评论1发你" | "感兴趣的朋友可以在评论区交流" |
| "私信领资料" | "有问题欢迎交流，看到都会回复" |
| "加我拿优惠" | "可以关注后续更新，有更多分享" |
| "主页有联系方式" | "会持续分享相关内容，欢迎关注" |
| "需要的扣1" | "大家有什么想法可以留言讨论" |
| "免费送资料包" | "整理了一些经验，分享给大家参考" |
| "点赞收藏不迷路" | "如果觉得有用可以留着备用" |
| "限时优惠，手慢无" | "目前还有活动，有需要可以了解一下" |

### 步骤五：CES评分优化与最终输出

#### 修改原则（新增）

所有修改必须遵循以下5条原则：

**原则1：降低风险**
- 绝对表达 → 相对表达（"最好"→"很喜欢的"、"绝对有效"→"对我挺有用的"）
- 承诺表达 → 体验表达（"保证变白"→"用了一段时间感觉有提亮"）
- 营销表达 → 场景表达（"赶紧下单"→"有需要的姐妹可以参考"）

**原则2：增强匹配**
- 让标题、开头、正文、标签都服务同一个目标人群
- 确保每个元素都指向同一类用户的需求和兴趣点
- 避免"既要又要"导致的人群模糊

**原则3：提升自然感**
- 减少官方口吻，增加真实使用语境
- 加入个人真实体验和细节描述
- 使用口语化表达，像朋友聊天一样自然

**原则4：保留核心卖点**
- 修改时不要过于保守，保留吸引力和转化价值
- 在合规的前提下，保留内容的"记忆点"和"传播点"
- 避免为了安全而牺牲内容的独特性和价值感

**原则5：给出可直接替换的版本**
- 每个关键问题都应给出可复制使用的替代表达
- 提供具体修改前后的对比，而非笼统建议
- 确保用户可以直接复制优化后的文案使用

#### CES评分模型（小红书内容质量核心指标）

CES（Content Engagement Score）是平台评估笔记质量的关键模型，优化方向如下：

| 指标 | 权重 | 优化策略 |
|------|------|---------|
| **点赞率** | 高 | 引发认同、提供情绪价值、内容有共鸣点 |
| **收藏率** | 高 | 提供实用价值、信息密度高、具备"以后再看"属性 |
| **评论率** | 高 | 设置互动钩子、提出开放性问题、引发讨论欲。注意：互动钩子需为具体问题，避免"扣1""评论区见"等无意义互动 |
| **转发率** | 中 | 内容有社交货币属性、适合分享给朋友 |
| **完播率** | 高（视频） | 前3秒抓人、节奏紧凑、信息密度适中 |

#### 网感强化要求

- **emoji合理**：标题和正文中适度使用emoji增强可读性，但避免堆砌
- **口语化表达**：使用"姐妹们"、"真的"、"谁懂啊"等小红书平台高频口语
- **情绪价值**：内容需传递真实情绪（惊喜、共鸣、治愈、愤怒、好奇）
- **UGC真实感**：避免官方腔调，像朋友分享一样自然，带个人真实体验细节
- **信息增量**：每篇笔记至少提供1个"原来如此"的信息点

#### 标题优化（输出3个高分标题备选）

基于原标题，提供3个优化后的高分标题备选：

- **标题A（痛点型）**：直击用户痛点，引发共鸣
- **标题B（好奇型）**：制造信息差，引发点击欲望
- **标题C（数字型/干货型）**：用具体数字或清单形式提升收藏率

标题要求：
- 长度12-20字为宜
- 包含核心关键词
- 避免敏感词
- 具备情绪张力或信息价值

  I. 其他

- **第二阶段**："您希望这篇笔记达到什么效果？" + 提供意图选项
  A. 涨粉导向（希望获得更多关注和曝光）
  B. 带货转化（推广产品/工具，引导下载/购买）
  C. 纯日常互动（分享日常，增强粉丝粘性）

- 根据补充信息完成完整检测流程

**第四步：用户不想补充**

- 基于文案文本进行推测性分析，但明确标注"基于文本推测"
- 仍提供优化建议，但建议用户补充信息以获得更精准的分析

## 风险等级定义

| 等级 | 分值 | 说明 | 处理建议 |
|------|------|------|----------|
| 🟢 安全 | 0 | 无敏感词，内容合规 | 可直接发布 |
| 🟡 低风险 | 1-3 | 少量轻微敏感词 | 建议微调后发布 |
| 🟠 中风险 | 4-6 | 较多敏感词或个别高风险词 | 必须修改后发布 |
| 🔴 高风险 | 7+ | 大量敏感词或严重违规词 | 严禁发布，需重写 |

## 注意事项

1. **定位先行**：务必先确认账号定位与人设，再进行内容检测
2. **意图明确**：每篇笔记必须有清晰的意图导向，避免内容策略混乱
3. **变体识别**：平台算法持续升级，需重点关注谐音梗、拼音缩写、emoji替代等隐蔽违规
4. **动态更新**：平台规则会变化，敏感词库需要定期更新
5. **人工复核**：AI检测可能有误判，重要内容需人工确认
6. **行业差异**：不同行业的敏感词标准不同，需针对性调整
7. **语境判断**：某些词汇在特定语境下可能不违规，需结合上下文
8. **平台差异**：不同内容平台规则不同，本 Skill 专门针对小红书
9. **CES导向**：内容优化需围绕CES评分模型，提升用户互动行为
10. **网感培养**：持续关注平台热门内容，培养对平台调性的敏感度
11. **不制造平台规则恐慌（新增）**：
    - 避免使用"必限流"、"一定违规"、"一定会被封"等绝对化表达
    - 使用"低风险"、"中风险"、"高风险"等分级表述
    - 使用"建议规避"、"建议替换"等建设性表达
    - 使用"需要结合账号历史数据判断"等客观表述
    - 强调"平台规则动态变化，建议持续关注官方更新"
    - 让用户理解风险可控，而非制造焦虑

## 最佳实践

### 长期运营建议

1. **保持定位一致性**：不要频繁更换内容方向，强化账号标签
2. **建立内容库**：积累合规的优质内容模板，形成SOP
3. **关注平台动态**：及时了解小红书规则更新与算法变化
4. **分析数据反馈**：根据笔记CES表现优化内容策略
5. **培养粉丝粘性**：真实互动，建立信任关系，提升评论率
6. **网感持续迭代**：定期分析爆款笔记，提炼网感要素
