# **敏感词替换建议库**

本文档提供常见敏感词、违规词及低质同质化词汇的合规替换方案，结合小红书最新社区规范（反焦虑、反照骗、AIGC规范），帮助创作者在保持表达效果的同时，避免违规或被系统限流。

## **使用说明**

### **替换原则**

1. **真实合规**：绝不触碰医疗、金融、导流等平台高压红线。  
2. **降级表达**：将“极限词”、“夸张词”降级为个人主观感受的“真诚分享”。  
3. **拒绝水文**：摒弃低质的网络黑话（如yyds），改用具体、有细节的感官描述。  
4. **符合人设**：替换词应符合账号的整体风格（如听劝型、专家型、闺蜜型）。

## **一、价值观与情绪合规（🚨 2026重点严打）**

小红书严厉打击“炫富”、“制造焦虑”及“极端对立”内容，此类词汇极易导致账号被封禁或严重限流。

## **四、旅游/探店用语替换（反照骗）**

平台严打“过度美化”和“滤镜照骗”，鼓励提供真实避坑指南。

| 原词（易遭举报） | 替换方案（真实/听劝） | 适用场景 |
| :---- | :---- | :---- |
| 国内版马尔代夫 | 原生态海岛、人少景美的小众沙滩 | 景点描述 |
| 美到窒息 | 真的很出片（建议配原相机直出图） | 视觉评价 |
| 此生必去 | 周末放松好去处、适合放空两天 | 推荐程度 |
| 吊打一切 | 各有特色、个人觉得比XX更安静 | 比较评价 |
| 避世仙境 | 商业化程度还不高（需说明交通不便） | 环境描述 |

**示例：**

* ❌ 绝美仙境！国内版马尔代夫，此生必去！  
* ✅ 原相机直出📸！人少景美的原生态海岛，但有3个避雷点...

## **七、增强 UGC 真实感的话术模板**

### **7.1 真实体验开场（拉近距离）**

* **时间背书**："空瓶记！用了整整两个月才敢来反馈..."  
* **自费声明**："自费购入，无广纯分享，说点大实话..."  
* **听劝人设**："听劝！上次发完笔记大家让我换个思路，于是..."  
* **原图声明**："原相机后置直出，无滤镜，瑕疵皮真实记录..."

### **7.2 坦诚表达缺点（建立信任）**

* "先说不足：包装有点简陋，而且油皮夏天用会略微有些厚重。"  
* "吐槽一下它的续航，实测只能用半天，但看在颜值的份上忍了。"  
* "不是完美的，如果你对XX要求极高，那这台机器不适合你。"

### **7.3 互动引导话术（合规提权）**

* **提问钩子**："有同样熬夜跨脸问题的姐妹吗？你们平时都怎么急救的？"  
* **经验交流**："你们一般买哪个颜色？在评论区给我抄个作业！"  
* **真实蹲守**："蹲一个用过两代的老用户，新款真的值得升级吗？"
