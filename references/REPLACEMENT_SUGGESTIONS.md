# 敏感词替换建议库

> 本文档提供常见敏感词的合规替换方案，帮助创作者在保持表达效果的同时避免违规。

---

## 使用说明

### 替换原则
1. **保持原意**：替换后的词汇应传达相近的意思
2. **符合人设**：替换词应符合账号的整体风格和定位
3. **自然流畅**：替换后语句应通顺自然
4. **避免堆砌**：不要过度使用替换词，保持内容真实

### 选择建议
- **专业型账号**：使用更专业、客观的表述
- **闺蜜型账号**：使用更亲切、口语化的表述
- **测评型账号**：使用更具体、数据化的表述

## 一、极限用语替换

### 1.1 "最"系列

| 原词 | 替换方案 | 适用场景 |
|------|---------|---------|
| 最好 | 很不错、挺喜欢的、个人很喜欢、感觉很棒 | 产品推荐 |
| 最强 | 很受欢迎、表现突出、实力很强 | 能力描述 |
| 最美 | 超好看、颜值很高、很上镜 | 外观评价 |
| 最棒 | 真的很赞、特别满意、超出预期 | 满意度表达 |
| 最爱 | 心头好、经常回购、一直在用 | 偏好表达 |
| 最优 | 很合适、性价比很高、值得推荐 | 性价比描述 |
| 最先进 | 技术很新、采用了新技术、很前沿 | 技术描述 |

**示例：**
- ❌ 这是我用过最好的面膜
- ✅ 这是我用过很不错的面膜 / 这款面膜我个人很喜欢

| 销量第一 | 销量很好、卖得很火、很畅销 | 销量描述 |
| 心中的第一 | 个人最喜欢的、特别喜欢的 | 个人偏好 |

**示例：**
- ❌ 这款产品销量第一

## 二、虚假宣传用语替换

### 2.1 夸大功效词

| 原词 | 替换方案 | 适用场景 |
|------|---------|---------|
| 立竿见影 | 效果明显、用了一段时间感觉不错 | 效果描述 |
| 瞬间 | 很快、不久、逐渐 | 速度描述 |
| 立即 | 很快、不久、过段时间 | 速度描述 |

| 颠覆 | 很不一样、和之前想的不同 | 差异描述 |

**示例：**
- ❌ 用完瞬间变白
- ✅ 用了一段时间肤色有提亮 / 坚持用效果挺明显的

| 原词 | 替换方案 | 适用场景 |
|------|---------|---------|
| 保证 | 应该、大概率、正常情况下 | 可能性描述 |
| 承诺 | 会尽力、目标是、希望 | 态度表达 |
| 无效退款 | 不满意可以联系客服 | 售后说明 |
| 治愈 | 改善、缓解、有帮助 | 效果描述 |
| 根治 | 有效改善、明显缓解 | 效果描述 |
| 痊愈 | 恢复得很好、改善很多 | 恢复描述 |

**示例：**
- ❌ 保证能治愈痘痘
- ✅ 对改善痘痘有帮助 / 很多用户反馈效果不错

### 2.3 医疗宣称词

| 原词 | 替换方案 | 适用场景 |

## 六、表达方式优化

### 6.1 主观感受表达

将绝对化表述改为个人感受：

| 原表述 | 优化表述 |
|--------|---------|
| 这是最好的产品 | 这是我个人很喜欢的产品 / 这款产品我用着很不错 |
| 效果立竿见影 | 我用了一段时间，感觉效果挺明显的 |
| 100%有效 | 我身边很多朋友用了反馈都不错 |
| 所有人都要买 | 有同样需求的姐妹可以考虑 |
| 必须入手 | 感兴趣的可以了解一下 |

### 6.2 数据化表达

用具体数据替代夸张描述：

| 原表述 | 优化表述 |
|--------|---------|
| 瞬间变白 | 用了2周肤色提亮了一个度 |
| 马上见效 | 坚持用了1个月看到效果 |
| 效果超级好 | 满意度90%以上 |
| 销量第一 | 月销量超过10000件 |

### 6.3 客观描述

用事实替代主观评价：

| 原表述 | 优化表述 |
|--------|---------|
| 最好的成分 | 含有XX成分，浓度是XX% |
| 顶级工艺 | 采用了XX技术/工艺 |
| 独家配方 | 特别添加了XX成分 |
