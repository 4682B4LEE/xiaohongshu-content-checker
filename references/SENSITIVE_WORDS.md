## 八、检测建议

### 8.1 变体识别
敏感词常有以下变体形式，检测时需要综合识别：

| 变体类型 | 说明 | 示例 |
|---------|------|------|
| **谐音梗** | 使用发音相近的汉字替换 | 薇信（微信）、票圈（朋友圈）、拼夕夕（拼多多） |
| **拼音缩写** | 使用拼音首字母或全拼缩写 | VX（微信）、GZH（公众号）、PYQ（朋友圈）、DDDD（懂的都懂） |
| **符号间隔** | 使用符号间隔敏感字 | 微❤️、V/X、W/信、加👆 |
| **emoji替代** | 使用emoji表情替代文字 | 薇❤️、二维🐴、抖🎵、快👋、🍑宝、🐶东 |
| **错别字** | 使用形近字或错误字 | 维信、威信、公粽号、拼夕夕 |
| **数字替代** | 使用数字或数字emoji替代 | 第1（第一）、1️⃣0️⃣0️⃣%（100%）、4️⃣信（私信） |
| **繁体字** | 使用繁体形式 | 薇信（微信）、獨家（独家） |
| **火星文** | 使用特殊字符组合 | 微★信、加☆我 |

### 8.2 检测策略建议
1. **组合检测**：单一变体可能无法识别，建议结合多种变体规则进行综合判断
2. **上下文分析**：某些词汇在特定语境下可能不违规，需结合上下文语义分析
3. **动态更新**：平台审核策略会不断演进，变体形式也在持续变化，需定期更新规则库
4. **阈值控制**：建议设置风险阈值，避免过度拦截正常内容
