# xiaohongshu-content-checker

基于多工作流的小红书内容风控 Agent，帮助创作者在发布前全面检测笔记合规性、优化内容定位、提升自然流量。

## 核心能力

### 1. 人设匹配与账号定位分析
- 分析账号人设类型与内容表达风格的一致性
- 识别人设漂移风险（如闺蜜型账号出现过度专业术语）
- 评估内容真实感与可信度

### 2. 敏感词排雷与违规风险检测
- 识别广告法极限词、违禁词、平台敏感词
- 检测变体规避手段（谐音、拼音、符号拆分等）
- 区分违规风险（可能触发审核）与流量风险（仅影响推荐表现）

### 3. 自然流优化与内容改写建议
- 评估营销浓度，识别硬广嫌疑和 UGC 感缺失
- 基于 CES 算法提供互动优化策略
- 生成可直接发布的合规改写版本

## 如何使用

推荐对话起手式：

```
请帮我检测这篇小红书笔记：
【账号信息】
- 账号类型：个人号/企业号/达人号
- 内容领域：数码工具/编程教程/效率办公/AI应用/生活方式等
- 账号阶段：新手期/成长期/成熟期/头部期
- 人设类型：干货型/测评型/情绪共鸣型/好物分享型

【笔记内容】
标题：...
正文：...
标签：...
封面文案：...

【笔记意图】
- 核心目标：涨粉导向/带货转化/纯日常互动/引流私域
- 目标关键词：...
```

如果信息不足，Agent 会先基于文本做基础检测，并引导补充关键信息。

## 适用人群

- **小红书新手博主**：建立正确的账号定位，避免违规踩坑
- **成长期创作者**：优化内容质量，提升账号权重
- **品牌运营人员**：确保商业合作内容合规
- **MCN 机构**：批量检测和优化达人内容

## 注意事项

1. **定位先行**：务必先确认账号定位，再进行内容检测
2. **动态更新**：平台规则会变化，敏感词库需要定期更新
3. **人工复核**：AI 检测可能有误判，重要内容需人工确认
4. **行业差异**：不同行业的敏感词标准不同，需针对性调整
5. **语境判断**：某些词汇在特定语境下可能不违规，需结合上下文
