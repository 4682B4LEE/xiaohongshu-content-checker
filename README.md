# 小红书笔记敏感词检测与内容定位分析 Skill

一个专为小红书创作者设计的智能内容检测工具，帮助你在发布前全面检测笔记合规性、优化内容定位、提升自然流量。

## 核心特点

### 🎯 先定位，后检测
不同于传统的敏感词检测工具，本 Skill 采用"先定位、后检测"的工作流程：
1. **账号定位分析** - 确认账号人设与目标受众匹配度
2. **内容方向检测** - 分析笔记内容与账号定位的一致性
3. **敏感词检测** - 识别广告法极限词、违禁词、平台敏感词
4. **自然流优化** - 识别影响自然流量的因素并提供优化建议

### 🔍 全面检测维度
- **广告法极限用语**：最、第一、顶级、绝对等
- **虚假宣传用语**：立竿见影、瞬间见效、100%有效等
- **平台规则敏感词**：引流词、诱导互动词等
- **行业特定敏感词**：美妆、减肥、金融、教育等行业专用词
- **内容质量风险**：标题党、搬运抄袭、低质内容等

### 👤 人设匹配度诊断
- 分析账号人设类型与内容表达风格的一致性
- 识别人设漂移风险（如闺蜜型账号出现过度专业术语）
- 评估内容真实感与可信度

### 📊 营销浓度评估
- 识别硬广嫌疑：过度产品堆砌、官方话术、推销感
- 检测 UGC 感缺失：缺乏真实体验描述、口语化不足
- 评估内容自然度，避免被平台判定为商业推广限流

### 🚀 CES 算法洞察
- 基于点赞、收藏、评论、转发、完播率的权重分析
- 分析内容对目标人群标签的匹配度
- 提供 CES（Community Engagement Score）优化策略

### 🖼️ 封面文案检测
- 封面文字字数控制（8-16字最佳）
- 一眼看懂：避免信息过载，突出核心卖点
- 避免堆叠卖点：单点突破优于多点罗列
- 检测封面是否存在违规词或极限词

### 💬 评论区互动引导检测
- 评论区导流风险：引导加微信、扫码、私信领资料等
- 诱导私信检测："评论1发你"、"私信领"等话术
- 诱导点赞收藏："求点赞"、"必回关"、"互粉互赞"等
- 提供合规的互动引导替代表达

### 🏭 特殊行业规则
针对不同行业内容的专项检测：
- **AI工具/SaaS**：避免过度承诺效果、技术门槛过高、缺少使用场景
- **医美/减肥/健康**：医疗承诺、疗效表达、药品推荐、前后对比风险
- **教育/副业/赚钱**：收益承诺、速成话术、制造焦虑、诱导报名
- **品牌种草/商业合作**：广告感过重、缺少真实体验、未披露合作风险

### 💡 智能优化建议
- 提供针对性的替换词汇建议
- 生成修改后的合规版本
- 分析账号定位与内容匹配度
- 提供自然流优化策略

## 适用人群

- **成长期创作者**：优化内容质量，提升账号权重

## 工作流程

### 步骤1：账号基础信息（选项式）
- 账号类型：A.个人号 / B.企业号 / C.达人号 / D.其他
- 内容领域：A.数码工具 / B.编程教程 / C.效率办公 / D.AI应用 / E.开源项目 / F.产品设计 / G.职场成长 / H.生活方式 / I.其他
- 账号阶段：A.新手期(0-1000粉) / B.成长期(1000-1万粉) / C.成熟期(1万-10万粉) / D.头部期(10万+粉)
- 账号IP/所在地（选填）

### 步骤2：人设定位（选项式）
- A.干货型（分享技术教程、使用技巧、经验总结）
- B.测评型（真实体验产品，客观评价优缺点）
- C.情绪共鸣型（分享开发故事、成长经历、心路历程）
- D.好物分享型（推荐实用工具、效率神器、数码好物）
- E.其他

### 步骤3：笔记内容与意图
- 提供笔记内容（标题/正文/标签/封面文案）
- 笔记意图：A.涨粉导向 / B.带货转化 / C.纯日常互动 / D.引流私域
- 目标关键词（1-3个）

### 步骤4：自然流阻碍因素排查
- **匹配度检测**：内容与人设、受众、赛道的匹配程度
- **营销浓度评估**：识别硬广嫌疑和 UGC 感缺失
- **导流风险检测**：排查引流到其他平台的风险
- **封面文案检测**：字数是否8-16字、是否一眼看懂、是否堆叠卖点
- **评论区互动引导检测**：是否存在诱导私信、导流、诱导点赞收藏

### 步骤5：文案像素级敏感词排雷
- 全面扫描标题、正文、标签
- 分类标注敏感词（极限词、违禁词、引流词等）
- **变体识别**：识别谐音、拼音、符号拆分等规避手段
- **规范输出**：按风险等级结构化呈现检测结果
- **区分违规风险与流量风险**：明确标注是可能触发审核，还是仅影响推荐表现

### 步骤6：最终优化与发布建议
- **修改原则**：
  1. 降低风险：绝对表达改相对表达，承诺改体验
  2. 增强匹配：标题、正文、标签服务同一目标人群
  3. 提升自然感：减少官方口吻，增加真实使用语境
  4. 保留核心卖点：降低风险的同时保留吸引力和转化价值
  5. 给出可直接替换版本：每个关键问题都提供可复制的替代表达
- **CES 优化**：基于互动数据的算法优化建议
- **标题优化**：生成 3 个高分标题备选
- **内容润色**：带 emoji、口语化、UGC 感的修改版本
- **9步标准化输出模板**：
  1. 总体结论（风险等级+是否建议发布+一句话判断）
  2. 内容定位卡（账号定位/目标人群/笔记类型/发布目的/核心内容/匹配度）
  3. 核心问题摘要（Top 3 问题）
  4. 敏感词检测（区分违规风险与流量风险）
  5. 账号定位与人群匹配度
  6. 自然流影响因素判断
  7. 修改建议（含修改原则）
  8. 优化版本（可直接发布）
  9. 发布前 CheckList

## 风险等级说明

| 等级 | 分值 | 说明 | 处理建议 |
|------|------|------|----------|
| 🟢 安全 | 0 | 无敏感词，内容合规 | 可直接发布 |
| 🟡 低风险 | 1-3 | 少量轻微敏感词 | 建议微调后发布 |
| 🟠 中风险 | 4-6 | 较多敏感词或个别高风险词 | 必须修改后发布 |
| 🔴 高风险 | 7+ | 大量敏感词或严重违规词 | 严禁发布，需重写 |

## 敏感词分类

### 1. 广告法极限用语（高风险）
- 最、第一、唯一、顶级、极致、绝对、永久、100%、零风险
- 首选、首家、独家、首发、首创、领先、王牌、冠军

### 2. 虚假宣传用语（高风险）
- 立竿见影、瞬间、立即、马上、神奇、奇迹、颠覆
- 保证、承诺、无效退款、治愈、根治、痊愈、永不

### 3. 平台规则敏感词（中高风险）
- 引流词：微信、VX、加V、二维码、私信我
- 诱导互动：求关注、必回关、互粉、互赞、求点赞
- 虚假互动：刷赞、刷粉、买数据

### 4. 行业特定敏感词
- 美妆：美白、祛斑、抗衰、祛痘、修复（需资质）
- 减肥：减肥、瘦身、燃脂、排毒、月瘦XX斤
- 金融：稳赚、保本、零风险、高收益、暴富
- 教育：保过、包过、包就业、100%就业、速成

## 最佳实践

### 分阶段提供信息建议

建议用户分阶段提供信息，降低填写负担，提升检测精准度：

1. **先提供账号基础信息**（4个选项）
   - 账号类型：A.个人号 / B.企业号 / C.达人号 / D.其他
   - 内容领域：A.数码工具 / B.编程教程 / C.效率办公 / D.AI应用 / E.开源项目 / F.产品设计 / G.职场成长 / H.生活方式 / I.其他
   - 账号阶段：A.新手期(0-1000粉) / B.成长期(1000-1万粉) / C.成熟期(1万-10万粉) / D.头部期(10万+粉)
   - 账号IP/所在地：可选，影响同城推荐权重不高

2. **再提供人设定位**（1个选项）
   - A.干货型（分享技术教程、使用技巧、经验总结）
   - B.测评型（真实体验产品，客观评价优缺点）
   - C.情绪共鸣型（分享开发故事、成长经历、心路历程）
   - D.好物分享型（推荐实用工具、效率神器、数码好物）
   - E.其他

3. **然后提供笔记内容**（标题/正文/标签）
   - 标题：笔记标题
   - 正文：完整正文内容
   - 标签：所有标签
   - 封面文案：封面上的文字

4. **最后确认笔记意图**（1个选项+关键词）
   - A.涨粉导向（希望获得更多关注和曝光）
   - B.带货转化（推广产品/工具，引导下载/购买）
   - C.纯日常互动（分享日常，增强粉丝粘性）
   - D.引流私域（引导到微信/社群/公众号）
   - 目标关键词：希望获得搜索流量的关键词（1-3个）

**分阶段优势**：
- 降低用户一次性填写负担
- 每个阶段可以及时获得反馈
- 便于根据内容分析调整后续意图
- 更接近真实创作场景（先写内容，再明确目的）

### 信息不足处理建议

如果用户只提供文案，未提供账号定位信息：

1. **先进行基础检测**：基于文本本身进行敏感词和文案质量检测
2. **最多询问3-5个关键问题**：
   - 这个账号主要做什么方向？
   - 这篇笔记想吸引哪类人？
   - 这篇笔记的目标是曝光、涨粉、咨询，还是转化？
   - 这篇笔记属于什么类型（种草/测评/教程/避坑）？
   - 账号之前主要发哪些内容？
3. **用户不想补充也能继续**：标注"账号匹配度判断信息不足"，基于文本推测完成检测

## 注意事项

1. **定位先行**：务必先确认账号定位，再进行内容检测
2. **动态更新**：平台规则会变化，敏感词库需要定期更新
3. **人工复核**：AI检测可能有误判，重要内容需人工确认
4. **行业差异**：不同行业的敏感词标准不同，需针对性调整
5. **语境判断**：某些词汇在特定语境下可能不违规，需结合上下文
6. **区分风险类型**：明确区分"违规风险"（可能触发审核）和"流量风险"（仅影响推荐表现）
7. **修改原则**：降低风险、增强匹配、提升自然感、保留核心卖点、给出可直接替换版本
